## Configuraciones

### Pre Requesitos

- Intalar [Node.js]

### Configuraciones Iniciales

Instalar de manera global el Angular CLI:

```
npm install -g @angular/cli
```

Clonar Projecto:

```
git clone http://github.com/pedroarauz1988/inventarioVacunasReto.git
```

Levantar la aplicacion:

```
cd [PROJECT NAME]
ng serve
```
